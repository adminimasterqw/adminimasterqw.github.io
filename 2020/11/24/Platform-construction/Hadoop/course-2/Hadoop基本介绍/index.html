# Hadoop 的介绍及相关知识

狭义上，Hadoop 就是单独指代 Hadoop 这个软件

HDFS：分布式文件系统

MapReduce：分布式计算系统

Yarn：分布式样集群资源管理

广义上，Hadoop 指代大数据的一个生态圈，包括其他

# 1、Hadoop 核心 - HDFS

1、介绍：大量数据跨机器存储，统一管理分布在集群上的文件系统，也就是分布式文件系统

2、架构

HDFS 是一个主 / 从（Mater/Slave）体系结构

HDFS 由四部分组成，HDFS Client、NameNode、DataNode 和 Secondary NameNode
