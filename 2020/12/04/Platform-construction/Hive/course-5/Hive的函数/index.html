# 2、LIMIT 语句

典型的查询会返回多行数据。LIMIT 子句用于限制返回的行数。

# 3、where 语句

# 4、LKE 和 RLKE

1. 使用 LIKE 运算选择类似的值
2. 选择条件可以包含字符或数字

3.RLKE 子句是 Hive 中这个功能的一个扩展，其可以通过 Java 的正则表达式这个更强大的语言来指定匹配条件。

4. 案例实操

操作符含义
AND逻辑并
OR逻辑或
NOT逻辑否
