# Flask 简介

# 介绍

Flask（http://flask.pocoo.org/）都算是小型框架，小到可以称为 “微框 架”，Flask 自开发伊始就被设计为可扩展的框架， 它具有一个包含基本服务的强健核心，其他功能则可通过扩展实现。你可以自己挑选所需 的扩展包，组成一个没有附加功能的精益组合，从而完全精确满足自身需求。

# 组成

Flask 有两个主要依赖：路由、调试和 Web 服务器网关接口（Web Server Gateway Interface， WSGI）子系统由 Werkzeug（http://werkzeug.pocoo.org/）提供；模板系统由 Jinja2（http:// jinja.pocoo.org/）提供。Werkzeug 和 Jinjia2 都是由 Flask 的核心开发者开发而成。

# 环境搭建

本章主要介绍 Pycharm 编辑器下的环境启动

文件 -> 新建项目 ->flask

虚拟环境使用第三方实用工具 virtualenv 创建。

检查是否安装

$ virtualenv --version

Linux 系统安装

$ sudo apt-get install python-virtualenv

MacOS 安装

$ sudo easy_install virtualenv
